# Python Blackjack Game

## Overview
This project is a command-line Blackjack game built in Python. It simulates a simplified version of the classic casino card game, allowing a user to play against a dealer.

The program uses object-oriented design and implements core game logic such as dealing cards, calculating hand values, and determining game outcomes.

## Features
- Player vs Dealer gameplay  
- Hit or stand decision system  
- Automatic dealer behavior  
- Hand value calculation with bust detection  
- Input validation for user actions  
- Multiple rounds of gameplay  
- Clear win/lose conditions  

## Technologies Used
- Python  
- Object-Oriented Programming (OOP)  

## How It Works
- A deck of cards is created and shuffled  
- The player and dealer are each dealt a starting hand  
- The player chooses to hit (draw a card) or stand  
- The dealer follows predefined rules to play their turn  
- The game determines the winner based on hand values  

## How to Run
1. Make sure Python is installed on your system  
2. Download or clone this repository  
3. Run the main Python file:

```bash
python BlackjackGame.py
```

4. Follow the prompts in the terminal to play
